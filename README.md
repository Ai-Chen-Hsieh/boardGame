## project發想
- 喜歡桌遊，所以以桌遊形象為主題的網站，練習以React開發&畫面切版，嘗試自行設計網站UI與簡單功能。

## 使用技術
 - React
 - react-router-dom
 - vite建置環境
 - styled-component
 - popup modal 製作
 - toastify 製作
 - axios 串接API
 - RWD
 - vercel部署

## 功能
主要分為兩區塊，最上面顯示熱門遊戲，下方顯示各類別的遊戲
-	點擊遊戲卡片會顯示遊戲的詳細資訊
-	遊戲資訊內購物車圖示可以將遊戲添加至購物車中
-	點擊遊戲卡片右上角的"+"圖示可以將遊戲收藏至最愛清單
-	遊戲搜尋功能
-	進入購物車可以檢視加入的商品
-	購物車內的商品數量可以點擊 + -圖示增減數量
-	購物車會自動計算總金額
